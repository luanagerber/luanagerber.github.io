Movido por Números

An interactive mathematical story book for iPhone

About the Project

Movido por Números ("Driven by Numbers") is my first iPhone app created independently. Developed in SwiftUI using MVC architecture, this interactive story book combines education with entertainment through mathematical problem-solving.

Follow young Eduardo's journey of personal development as he searches for his professional path. Users must solve mathematical expressions to generate fuel for a spaceship, symbolizing Eduardo's quest for self-discovery.

How to run

  1. Clone the repository from GitHub to Xcode
  2. Open the project in Xcode on your macOS
  3. Run with Xcode iPhone Simulator
  4. Start solving math problems and progress through Eduardo's story!
  5. Experience the journey of personal and professional discovery
